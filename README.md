## Usage

### How to use the website

#### Navigating to the website

To use this schedule simply navigate to https://samiboi1.github.io/valar_pa_land/.

#### What can I do?

On the website you can see when and where Valar på Land have concerts and a few pictures from photoshoots and concerts from the past

## Contributing

Contributions are what make the open source community such an amazing place to learn, inspire, and create. Any contributions you make are **greatly appreciated**.

If you have a suggestion that would make this better, please fork the repo and create a pull request. You can also simply [open an issue](https://github.com/Samiboi1/valar_pa_land/issues) with the tag "enhancement".
Don't forget to give the project a star! Thanks again!
